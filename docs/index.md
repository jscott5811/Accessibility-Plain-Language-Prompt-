# Accessibility and Plain Language Prompt Toolkit

A toolkit for producing accessible content by translating complex text into two clear formats: Plain Language and Easy Read. Designed to help writers, editors, and AI systems produce materials that are easier to understand on the first read.

## Quick start

1. Copy the contents of `a11y-plain-language-v1.txt` into the System Instructions or Custom Instructions field of your LLM.
2. Provide the source text and tell the model to output either "Plain Language" or "Easy Read".
3. Review the generated output with a human reviewer who has accessibility expertise.

See the repository README for complete usage instructions and examples.
