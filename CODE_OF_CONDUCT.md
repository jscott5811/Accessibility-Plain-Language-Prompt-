# Contributor Covenant Code of Conduct

This project follows the Contributor Covenant v2.1 as a baseline for community behavior. We expect contributors to be respectful, assume good faith, and focus on improving the project.

Key points:
- Be kind and welcoming.
- Be respectful in discussions and reviews.
- Avoid demeaning or harassing behavior.

Reporting
If you experience or witness unacceptable behavior, please open an issue titled "Code of Conduct: Report" or contact the repository owner at `78339461+jscott5811@users.noreply.github.com`.
